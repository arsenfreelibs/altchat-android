alt-chat — Политики пользователя

На этой странице изложены основные политики и соглашения, регулирующие использование платформы обмена сообщениями alt-chat.

1. Пользовательское соглашение и политика конфиденциальности

Используя alt-chat, пользователи соглашаются соблюдать настоящие условия.

Обязанности пользователя

  • Использовать сервис только в законных целях
  • Не распространять незаконный или вредоносный контент
  • Не заниматься мошенничеством, рассылкой спама или распространением вредоносного ПО
  • Уважать права других пользователей и третьих лиц

Обработка данных

Сервис может обрабатывать ограниченный объём технической информации, необходимой для работы: номер телефона, имя пользователя, информацию об устройстве, IP-адрес и журналы соединений.

Данные используются для обеспечения работы сервиса, безопасности, предотвращения мошенничества и технической поддержки.

Запросы правоохранительных органов

Данные пользователей могут быть переданы уполномоченным органам только при наличии соответствующего законного требования.

2. Модерация контента

Пользователи могут сообщать о контенте, который: нарушает действующее законодательство, содержит мошенничество или обман, распространяет вредоносное ПО, нарушает права интеллектуальной собственности, содержит угрозы или преследование.

Меры по модерации могут включать предупреждение, ограничение, удаление контента или постоянную блокировку аккаунта.

3. Соответствие требованиям к приложению

alt-chat — платформа для законного обмена сообщениями: текстовые сообщения, обмен файлами, голосовая связь, групповые чаты.

Сервис предоставляется «как есть» без гарантии бесперебойной работы.

4. Запросы правоохранительных органов

Запросы рассматриваются только при условии их подачи уполномоченными органами и соответствия действующему законодательству.

Официальные запросы: legal@alt-chat.me

5. Антиспам-политика

  • массовая рассылка незапрошенных сообщений запрещена
  • фишинговые и мошеннические ссылки запрещены
  • массовая реклама без согласия получателей запрещена

Аккаунты, нарушающие правила борьбы со спамом, могут быть ограничены или заблокированы без возможности восстановления.

Сообщить о спаме: support@alt-chat.me
